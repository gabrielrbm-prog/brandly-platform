Iconografia

        
Utilizamos Lucide Icons como biblioteca padrão. Ícones devem ser usados em 16px, 20px ou 24px com stroke-width de 2.


        

            

                
16px - Inline com texto, badges

                
20px - Sidebar, botões, inputs

                
24px - Headers, cards, destaques

                
32px - Apenas decorativo

            

            

                npm install lucide-react | CDN: https://unpkg.com/lucide@latest
            

        

    


    
    

        
Animações

        
Diretrizes para transições e animações consistentes em toda a plataforma.


        

            

                
Durações

                

                    
Micro (hover, toggle)150ms

                    
Rápida (fade, slide)200ms

                    
Normal (modal, drawer)300ms

                    
Lenta (page transition)500ms

                    
Dramática (onboarding)800ms

                

            


            

                
Easing Functions

                

                    
Padrãoease-out

                    
Entradacubic-bezier(0.4, 0, 0.2, 1)

                    
Saídacubic-bezier(0, 0, 0.2, 1)

                    
Bouncecubic-bezier(0.68, -0.55, 0.27, 1.55)

                    
Spring (GSAP)elastic.out(1, 0.3)

                

            


            

                
Padrões de Entrada

                

                    
Fade In: opacity 0 → 1, 200ms ease-out

                    
Slide Up: translateY(20px) → 0, 300ms ease-out

                    
Scale In: scale(0.95) → 1, 200ms ease-out

                    
Stagger: Delay de 50ms entre itens de lista

                

            


            

                
Padrões de Saída

                

                    
Fade Out: opacity 1 → 0, 150ms ease-in

                    
Slide Down: translateY(0) → 10px, 200ms ease-in

                    
Scale Out: scale(1) → 0.95, 150ms ease-in

                    
Collapse: height auto → 0, 200ms ease-in
